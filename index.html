General info

Website is a WIP.

This webpage serves the purpose of showing and visualizing Counter-Strike: Global Offensive case/capsule/package odds, in order to give the user a better overview of the actual odds of getting an item of sought rarity.

The data shown on this webpage is based on a blog post from 2017 by Perfect World. The same data is used for cases/capsules/packages released post-2017.
